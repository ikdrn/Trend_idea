# 20260303 設計メモ — trend-mixer-cli（再作成版）

## 1. 目的
トレンド情報を読むだけで終わらせず、
「次に作るもの」を短時間で決められる形に変換する。

## 2. 入力
- `--trend "名前|課題|実装方針"` を2件以上
- または `--demo`（サンプル入力）
- 任意で `--pattern`（出力テンプレート切り替え）

## 3. 出力
- システム名
- キャッチコピー
- 解く課題
- 実装方針
- 1〜3時間で作る最小ステップ

## 4. 技術選定
- 言語: JavaScript（Node.js）
- 理由:
  1. JS/TS ランタイムの注目トレンドを反映
  2. 実行環境に Node.js があり、即実行できる
  3. 依存ゼロで初学者が再現しやすい

## 5. 最小機能
1. 引数を手動パース（`process.argv`）
2. `--trend` 文字列を `|` で分解
3. テンプレートから提案文を生成
4. コンソールに表示

## 6. 今後の拡張
- `--save` で Markdown 保存
- 入力検証の強化（重複チェック）
- JSON 出力モード（CI連携向け）
